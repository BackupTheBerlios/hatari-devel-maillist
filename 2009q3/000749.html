> I'd like to use the portaudio library

> to emulate the jack entrance of the falcon.

Do you need the whole portaudio or only small part of it?


> This library works on linux, windows and mac. It's free.

The license seems to be GPL i.e. Hatari license compatible.


> The main problem is that I would have to include it completely
> (as it is done with audacity for example).
>
> Do you agree for this ?

I assume you don't need a bleeding edge version from Portaudio SVN.

The distribution version of portaudio may be newer than the Hatari one &
have distro specific fixes. We don't want to deal with Portaudio problems,
it's better that whoever provides portaudio to Linux distro / Windows / OSX,
handles that.

So, if it's included into Hatari sources, it should be integrated in a way
that the sources get used only if configure doesn't find this library from
the system.

(And of course its license needs to be mentioned in documentation where
we list other copyrights etc.)
