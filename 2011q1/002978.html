Hi,

just a quick bug report:

When using Hatari in fullscreen mode with a dual head setup, the screens 
get mirrored and when exiting back from Hatari fullscreen the screens are 
still mirrored. One needs to go to the monitor setup and re-configure it 
to have the normal dual-head setup back.

Machine: HP dc5800 with Intel graphics
OS: Ubuntu 10.10 with all updates
Hatari: Latest hg

In my opinion this is another reason why changing resolution isn't a good 
idea; a hardware scale to the native resolution would eliminate this 
problem, and it would also allow a fullscreen Hatari run on one screen, 
and a devkit (in my case Eclipse / Vasm) on the other. Using (SDL)OpenGL 
backend would make it possible to use hardware scaling.

Yeah I know, I culdn't help plug that OpenGL would benefit Hatari, and not 
just for OS X ;-)
