Use the source, Luke! ;-)

Have a look at src/gui-win/opencon.c :

void Win_OpenCon(void)
{
        if (!bExceptionDebugging)
                return;

        AllocConsole();
        freopen("CON", "w", stdout);
        freopen("CON", "r", stdin);
        freopen("CON", "wr", stderr);
}

... and bExceptionDebugging is set by the "-D" option.
