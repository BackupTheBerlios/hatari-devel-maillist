Yes, but please, one problem at a time :)

Let's first make Falcon work with its genuine specification, and once 
it's done we can see what it needs to support fast ram or other 
expansion cards.

We can think about it earlier of course, but I think it would be an 
error to commit code for this now, there's too much things that still 
don't work for Falcon.

It's only when we have a working/solid falcon emulation that we can be 
sure to have some good basis for further cpu.

Let's not go in too many directions, better do some incremental 
improvements :)
