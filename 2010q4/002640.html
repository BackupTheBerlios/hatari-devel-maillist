It crashed at the first Exception ... there was some Amiga specific
code in Exception_normal() which did not work in Hatari:

 	if (nr >= 24 && nr < 24 + 8 && currprefs.cpu_model <= 68010)
 		nr = x_get_byte (0x00fffff1 | (nr << 1));

I've disabled it and did two other small fixes, and now it is booting
up to the desktop :-)
