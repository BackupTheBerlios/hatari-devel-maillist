A few years ago, I had the feeling that uae-jit was more limited than 
the normal cpu core and some programs couldn't run when jit was used.
Porting uae-jit into hatari would not be possible I think, this is 
another code branch than normal uae core, with different patches that 
would need to be adapted again. Quite a lot of work.

I think the only way to go is to integrate winuae, this is the only 
actively maintained 68000's cpu for amiga/atari type of computers.
