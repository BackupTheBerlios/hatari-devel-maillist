I believe the Desktop idling at 79% (above) is meaningless because
my cpu was max'd-out. Better Falcon Desktop info is with:

"Frame Skip" = 1
"DSP" = off

"Frame Skip = auto" looks like the best option

Result with Blue: (Hatari = 86%CPU), (Xorg = 12%CPU) (Cpu = 96%us max'd)
