Hi,

I noticed a bug with Hatari's fire button while reading some atari-forum
posts. The Downfall game requires the fire button to be pressed to start
the game. If the button is pressed for more than one frame the game will
freeze, thus, the button must be tapped quickly. When the game goes to
the next round the button must be successfully tapped quickly again to
continue.

This problem existed with early versions of Steem and was solved.
Maybe Hatari could have this one looked at before the next version. :)

Downfall:
http://www.atarimania.com/game-atari-st-downfall_9157.html


 Here is the story:

*Steem Authors: (Nov 24 2003 - Regarding Steem)
http://www.atari-forum.com/viewtopic.php?f=4&t=1604&p=9711&hilit=Downfall&sid=db09b862bb7d2cd90be76e390556d81a#p9711

Downfall freezing up is actually a Steem bug, some silly keyboard controller
speed thing, we will sort it eventually. To get round it currently tap the
button as quickly as possible, if you still have problems then turn on slow
motion as you do it, that will guarantee the button will be held down for
only one frame.

In the game you can enable more of the options up the top by directing snow
into the channels at the side of your area.
