LAURENT SALLAFRANQUE wrote:
> What is mouse grab supposed to do (AltGr + M)? 
> I've done the test under windows, but I detected to change.

There is a problem with SDL and the AltGr key on Windows (not Linux) 
with some keyboard layouts. It affects the French keyboard, and also the 
German keyboard if I'm not wrong.
On these keyboards, SDL detects the AltGr key as Control+AltGr, so the 
Hatari keybindings probably fail.

It works for me if I remap the key to something else.
