Instructions can be found from the top of the configure.ac file:
# To create all necessary files for configure, use:
#   aclocal && autoheader && automake --add-missing --copy && autoconf
# Later on you can use "autoreconf -i" after updating this file.

(I look that up from there too :-))


Another option is just to use the Makefile-default.cnf config-default.h
as templates and edit them manually.
