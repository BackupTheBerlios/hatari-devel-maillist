don't know yet what's wrong with ACSI, but there is a bug in the debug
output in hdc.c, line 566:

	fprintf(hdlogFile, "LBA: 0x%lx\n", HDC_GetOffset());

This should be:

	fprintf(hdlogFile, "LBA: 0x%lx\n", HDC_GetOffset()/512);

Otherwise not the logical block address but the offset in the image file
is displayed.
