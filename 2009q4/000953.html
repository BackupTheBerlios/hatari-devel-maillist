> What is 11 bombs error ?
> 
> That's what I get with multi brique game.
> 

11 bombs mean illegal Line-F instruction. Line-F instructions are used
for the FPU and MMU in the Falcon. So if you run with plain 68020, try
to enable the "+FPU" mode. If it still crashes with 11 bombs, that most
likely means that the game tries to re-program the MMU. So this program
won't work with current Hatari since we don't have MMU emulation yet.
