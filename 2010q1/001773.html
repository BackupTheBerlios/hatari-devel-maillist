> CMake doesn't work on my Mac.

It does on mine. This is how I do it: Install macports, install
mercurial, cmake , and libsdl using macports.
Get hatari sources from mercurial and run:
./configure --disable-osx-bundle
make

No, you don't get the fancy gui.
I also created a port of hatari to macports so you can install the
latest stable version straight from macports :)
