What you see is a color change of the background, not actual pixel/bitmap.
If the demo changes colors at cycle 480 instead of 0 for example (which is 
needed since cycle 0 is used for the left border removal), you get this 
effect because color is indeed changed one line earlier.

You would see the same thing on a real ST if you set your monitor to 
display the full right area of the screen.
