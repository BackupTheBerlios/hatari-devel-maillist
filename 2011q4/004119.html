Dear Hatari users,

EmuTOS 0.8.6 is about to be released.

I have built a beta version from the current CVS sources. It is available here:
http://vincent.riviere.free.fr/soft/m68k-atari-mint/archives/unsupported/emutos/cvs/

The documentation has not been updated yet, but the ROMs are almost ready 
for release.

The main features are:
- Big Partition Support: allow FAT16 partitions up to 2 GB
- LMC1992 initialization on STe/TT hardware

I suggest you to try the new ROMs as a replacement for the old ones on your 
favorite Hatari setups, and just check it still works as usual. You should 
not encounter any trouble.

However, if you notice any regression, please report it preferably to 
emutos-devel at lists.sourceforge.net as soon as possible so we will fix that 
before the release.

Thanks in advance.
