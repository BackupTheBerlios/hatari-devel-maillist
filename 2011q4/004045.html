No need to change the CMake files here, just set the CC environment
variable to clang before running cmake:

CC=clang cmake .

> > Have you got another idea I should investigate to solve this
> > problem ?

You could also save the pre-processor output by adding -save-temps to
the CFLAGS:

CFLAGS="-save-temps -O2 -g" cmake .

When you now recompile the file, there should be a .i file in the same
directory that shows you the output of the pre-processor.
