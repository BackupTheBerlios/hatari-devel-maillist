Attached is an updated patch.

It centers the ST/E screen to the desktop resolution, I tested that it
works with the VDI mode and unlike previous patch, it shows statusbar also
in fullscreen and doesn't cause statusbar assert if one does a ST reset
in fullscreen...

Additionally, if you use "-z 1" in windowed mode, it will use the 2x zoom
for the ST/E screen in fullscreen instead of 1x zoom if there's space for
2x screen.
