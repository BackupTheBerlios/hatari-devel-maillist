cycInt.c uses stdint.h.

SDL types were in use by the Hatari code before we "switched"
to C99 that provides stdint.h.

But Hatari has also a lot of code coming from many different places,
see the end of the doc/authors.txt file.

For code that originates in Hatari it makes sense to use the SDL types
(as those are used also elsewhere in Hatari) where applicable.
