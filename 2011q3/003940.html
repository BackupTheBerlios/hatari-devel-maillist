I think I found what's the problem.

util-linux (= sfdisk, cfdisk, fdisk) sources have common code for checking
block device size, but on image files it doesn't fall back to using stat.
Then all of these stupid utilities complain that partition size is larger
than the maximum (=0 for image file).

-> loopback is probably a better option until util-linux is fixed.
