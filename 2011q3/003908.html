I've added a "fast boot" option now - when disabled, the EmuTOS splash
screen now shows up (you need also <= 4 MiB RAM for this because EmuTOS
can not detect more RAM without memvalid patching).

The "fast boot" option also works fine with TOS 2.06, so when you
disabled it, you can watch TOS' boring memory test bar there now, too.
