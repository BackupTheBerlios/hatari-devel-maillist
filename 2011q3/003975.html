By the way, there is a better way to get even more precise limits. For
each version of TOS HDDRIVER (like any other XHDI compatible driver)
maintains a list of the GEMDOS limits, like maximum sectors per cluster
etc. HDDRUTIL displays this limits when selecting the "GEMDOS Limits"
menu entry.
