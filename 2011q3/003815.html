Yes.
The main spirit of the GPL is: if you get some binary somewhere, you must 
also be able to get the corresponding sources.

After further checking, it appears that the tos.img file shipped in 
hatari-1.5.0_windows.zip is actually the official EmuTOS 0.8.5 binary 
etos512k.img which has been renamed.

Since this is the official EmuTOS 0.8.5 binary, I agree it is sufficient for 
Hatari to point to the official EmuTOS 0.8.5 sources on the EmuTOS website, 
as Eero suggests (as long as the EmuTOS website and the old source releases 
does not disappear - which should not happen).
