Well, readline is a library mainly used to handle keyboard shortcuts, 
history, moving into the line with various method, completing words, ... 
all those things that are pretty useful when you build a console / text 
interface for a program and want the user to be able to navigate easily 
through everything he typed.

Most famous readline usage is certainly the bash shell, which heavily 
really on realine functionnalities.

On ubuntu, try to install libreadline5-dev
