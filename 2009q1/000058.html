Hello

in order to update your Hg copy, you need to run :

hg pull https://npomarede@hg.berlios.de/repos/hatari

(from your hatari/ directory you cloned earlier)

Then you will need to run 'hg update' and perhaps 'hg merge' if you made 
some changes on your side and you want to merge everything.

Finally, 'hg push https://npomarede@hg.berlios.de/repos/hatari' will 
commit your changes/merges to berlios main repository.
