Like with CVS, the commands you listed work between the repository
you're using (the local one) and your working directory.


> I tried with MacMercurial, without any success.
>
> I can see that you made change to the Mercurial repos, but cannot get
> them...

To sync between repositories (like the BerliOS one and your local one), you
should use the hg "pull" and "push" commands.  To view the modifications
these commands will do, use the "incoming" and "outgoing" commands.

$ hg incoming -p | less
$ hg pull -u

I suggest reading the hg documentation and printing out one of the qref 
charts about the hg workflow/commands.
