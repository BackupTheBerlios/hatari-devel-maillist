That's already the case. Unless there is a bug somewhere ;-)
First, increase the log-level in your hatari.cfg file to 5, then the
emulator will inform you about floppy disk changes in the console
window like this:

Floppy A: has been removed from drive.
Inserted disk '/home/thomas/test1.msa' to drive A:.

When you see the "Floppy ?: has been removed from drive" that means
that the current floppy has been saved to the disk. See
Floppy_EjectDiskFromDrive() for details.
