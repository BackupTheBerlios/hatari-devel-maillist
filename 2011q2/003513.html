Looking at the beginning of that file with a hexeditor, it does not
seem to be a standard .msa image since it does not start with the magic
bytes 0x0e0f.
The images have likely been created with one of the many MSA clones,
which use the same file extension but a completely different file
layout.
So I suggest that you try to open the file with JayMSA instead, as
suggested on the webpage that Thomas Reiss mentioned.
