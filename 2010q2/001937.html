CMake should really use LINKER flags also when linking its test things.
And it might e.g. later leave CFLAGS out from linking of them, so I think
it's best to recommend using both:
 -DCMAKE_C_FLAGS=-lncurses
 -DCMAKE_EXE_LINKER_FLAGS=-lncurses

I updated the readme accordingly.
