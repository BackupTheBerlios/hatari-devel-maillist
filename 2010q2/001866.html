Hi,

I just wanted to inform that the hg repo version of Hatari does not build 
on Xcode without modifications.

I've done this:

1. hg clone https://developername@hg.berlios.de/repos/hatari
2. cd hatari
3. cmake -G Xcode .
4. Start the project file created in Xcode
5. Select release target
6. Hit build button

These are the three errors I get:

~/hatari/src/gui-osx/PrefsController.m:606: error: 'CNF_SCREEN' has no member named 'bZoomLowRes'
~/hatari/src/gui-osx/PrefsController.m:785: error: 'CNF_SCREEN' has no member named 'bZoomLowRes'
~/hatari/src/gui-osx/SDLMain.m:446: error: 'poseAsClass:' is unavailable (declared at /Developer/SDKs/MacOSX10.6.sdk/System/Library/Frameworks/Foundation.framework/Headers/NSObject.h:127)

To make it run thru compilation I simply comment out those three lines, 
which can't be an ideal solution.

System: Mac OS X 10.6.3
Xcode: 3.2.1 64-bit
