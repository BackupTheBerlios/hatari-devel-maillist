Hi,

 From my point of view, hatari is no more experimental for falcon emulation.
I would say that hatari is a falcon emulator witch allows 70-80% of 
programs to work.

Dsp let's run many programs and demos.
Dsp sound effects are working with winrec and whip.
Crossbar let run demos which use the SSI port.
There may still be 1 or 2 bugs in the dsp code, but they're harder to 
detect now.
Microphone entry is emulated too.

The "still to do" list is :

- precise Videl emulation
- verify dsp cycles
- better accuracy between DSP, crossbar and 68030
- emulate the non documented ssi transfert which is used in GT2 modules 
(noring's player).
- allow the crossbar to "distribute" sound in handshake mode to the DSP 
and non handshake mode to the DAC (for example).

So, don't hesitate to modify hatari's files and manuals.


Ps : I'll update the TODO list, as I've seen some things are already done.

Regards

Laurent
