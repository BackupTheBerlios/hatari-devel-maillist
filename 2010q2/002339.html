> > But why "+1". My information is that sysbase+0x0c is the first byte

>
> Both are correct !
>
> The memory contains "OS RAM" and just after there is "RAM not used from

Well, I didn't want the name to be too long so that values align nicely:
-----
> info osheader
OS base addr : 0xe00000
OS RAM end+1 : 0x008f9a
...
------

The "basepage" info uses similar heading:
------
> info basepage
Process basepage information:
- TPA start      : 0x00fce4
- TPA end +1     : 0x00fe64
...
------
