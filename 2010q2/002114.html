Hi,

after a quick test, it appears to work, there are few things which are 
inconsistent with the SDL GUI though.

1. Slow down FDC is under System in the OS X GUI and under Floppy disks in 
SDL GUI

2. The OS X GUI screen setup is missing the max zoom resolution from the 
SDL GUI, see screenshots compared: http://ae.dhs.nu/pics/hatarizoomsetup.png

Furthermore the SDL GUI have split up disk settings into two different 
menus, same with screen setup. But I guess this is due to limitations of 
the SDL GUI (size).

As a sidenote, the max width for "Max zoomed" is limited to 1280 and 
height to 960, which by todays standard feels a bit low (2560x1440 is 
becoming quite popular for example).

Anyways good to see that the OS X GUI is catching up :)
