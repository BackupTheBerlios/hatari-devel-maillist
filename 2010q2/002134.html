CMake supports Frameworks. Have a look at FindSDL.cmake from the CMake distribution for example and you'll see that they explicitely search for the SDL Framework there.
So please, take some time to read the user-friendly documentation on www.cmake.org, so you can integrate your changes to the standard CMake files of Hatari. Maintaining an additional Xcode project in the repository only resulted in problems in the past (the file list was never up-to-date, the CFLAGS often did not match, different versions of Xcode introduced incompatibilities, etc.).
