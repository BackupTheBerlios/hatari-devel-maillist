Things I noticed in the SDL changelog:
	http://www.libsdl.org/release/changes-1.2.html

1.2.14:
+ Fixed audio quality problem when converting between 22050 Hz and 44100 Hz.
+ Unix:
  - Updated ALSA support to the latest stable API
  - ALSA is now preferred over OSS audio.
    (SDL_AUDIODRIVER=dsp will restore the previous behavior.) 
  - Improved support for PulseAudio

1.2.12:
+ Fixed buffer overrun crash when resampling audio rates.

1.2.11/Windows:
+ DirectSound audio quality has been improved on certain sound cards.


I guess to try ALSA on CentOS, David would need to use:
	SDL_AUDIODRIVER=alsa hatari
