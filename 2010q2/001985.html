Eero Tamminen wrote:
 >> This sound delay is the first thing I have noticed when I ran Hatari for
 >> the first time, and it is one of the main reasons I prefer using Steem,
 >
 > What other reasons there are? :-)

Steem has an extremely nice GUI, especially for inserting floppies.
Hatari has ugly internal windows, with unfriendly file browsing.

Hatari default mouse behaviour (ungrabbed and unsynced) is very annoying.

Steem has very intuitive memory snapshots accessible from the toolbar.

Hatari has nice features, but hidden under obscure keystrokes.

These are the main points.
If I had to choose, I would say the biggest issue is the sound delay. I love 
to hear keyclicks on key down. Probably one of the best oldschool features.

However, Steem is a closed software, not updated anymore, thus sentenced to 
death. How sad for such a good piece of software.

On the other side, Hatari is Free Software, portable, and has more and more 
features, and is becoming more and more accurate, including Falcon hardware. 
So Hatari will live forever, and will always become better and better.
