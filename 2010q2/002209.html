> Hello,

> I've tested the latest syntax demo called Solar system. (found on DHS
> site) I've tested in Ste emulation mode.
> It seems to work well, except that pictures flickers a lot (is it the 
> same on the real hardware?)

That's a "normal" effect of some slideshows / demos. They slightly
switch the color palette from one frame to the other to simulate more
colors on the screen than a real ST / STE can show. On a normal ST
which is hooked to a TV or RGB monitor, the colors then "mix" on the
screen, but on a modern LCD screen, running on an emulator, you still
can see each frame separately, so you can see the flickering effect
there.

> In the "product info" screen, rasters are not stable on the left.

That might also happen on a real ST, you just normally don't see such a
big border there, so you would not notice the color switching on the
left.
