Googling found couple of possible ways to get Hatari launched with Terminal.

Either create a shell script that runs Hatari and add an icon for that:
	http://www.macosxhints.com/article.php?story=20010827215428108

Or create a Term file that runs Hatari:
	http://macdevcenter.com/pub/a/mac/2002/03/26/terminal_four.html

"Things really become interesting, however, when you take advantage of a 
little known feature of term files, the ability to define a command line to 
run when the window opens. Doing this allows you to create your own 
double-clickable run files that open in Terminal within a window of your 
exact specifications and run any command line (or script file) you can come 
up with."


	- Eero

PS. Here are some words about command line for OSX users:
  http://linuxdevcenter.com/pub/a/linux/2001/11/15/learnunixos.html

Some things are just so much easier on command line.
