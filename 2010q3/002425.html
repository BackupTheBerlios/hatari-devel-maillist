Hi,

On Wednesday 22 September 2010, Dima Sobolev wrote:
> I tested Falcon TOS 4.00 with Hatari. I heard that such a ROMs are
> multilanguage.

I quickly checked it.  TOS4 binary seems to contain texts in:
- English
- German
- French
- Spanish
- Italian


> Do you know how to switch to another language under Hatari ?

There's currently no option for that, so for now you either need to
edit the TOS binary itself or the ~/.hatari/hatari.nvram file.

I guess Hatari could like Aranym have an option to set it in NVRAM.
Does anybody on the devel list have any comments on this?


> And one more question. Can we go to Internet with Hatari ? It would be
> interesting to surf some pages with CAB.

Hatari doesn't emulate any network adapter, but I guess you could
do network connection through the serial or parallel emulation if
you have suitable Atari software for that inside the emulation and
on your host computer.
