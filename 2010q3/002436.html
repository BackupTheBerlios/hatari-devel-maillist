It was ported vom WinSTon, right?

I started writing on a state machine for the FDC, which handles thinks like different data rates, spin-up time, settle time for head movements, etc. and all the standard FDC commands. I also plan to support fuzzy sectors (sectors with wrong timing, that creates unstable bits). For simple disk formats it should still work with read track (it just puts the sectors into a "formatted" track), for Pasti files it hopefully does better :-) This will take a bit time (strangely I have a full-time job, too :-), I'll keep you updated.
