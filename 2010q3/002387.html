Hello

as reported on another forum, the part of StSound's code to handle noise 
(that was still present in Hatari 1.4) was discarding all noise per < 3, 
which resulted in no sound being output in that case.

I rewrote tone and noise step compute to remove StSound's remaining code 
in that case and to use the same logic as the one I used to compute env's 
per.

Old code is still present in case some regressions should be tested, but I 
will remove it soon.

If you ever noticed some different sound when noise was used, could you 
please test again with this new version (the maxymiser song "chipping 
for ca$h" was reported as such ; could someone test it now ?)
